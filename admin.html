📖 काम कैसे करें?

1. दुकान ID डालें - अपनी दुकान की ID लिखें और Enter दबाएं

2. सामान Edit करें - प्रोडक्ट के सामने ✏️ बटन दबाकर नाम/कीमत/फोटो बदलें

3. सामान Delete करें - 🗑️ बटन दबाकर सामान हटाएं

4. पूरी दुकान Delete - नीचे "दुकान हटाएं" बटन से

⚠️ दुकानदार को लूटें नहीं! केवल ₹20 ही लें।
